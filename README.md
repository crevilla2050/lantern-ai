# Lantern AI

An AI-powered terminal companion for developers and system administrators.

Lantern helps you understand your systems by illuminating logs,
diagnosing failures, explaining commands, and gathering evidence
before action is taken.

## Philosophy

See first. Act second.

## Features

- Explain logs and errors
- Interactive AI shell
- Unix pipe support
- Session management
- Multiple AI providers
- Diagnostics and troubleshooting
- Plugin architecture

Early development (v0.1).
